# Keybindings Documentation

A quick-reference guide for keyboard shortcuts and commands across operating systems, window managers, editors, terminal tools, browsers, and creative applications.

## Operating Systems

### Windows

- **[Windows](windows.md)** -- General shortcuts, File Explorer, Task Manager, Virtual Desktops
- **[GlazeWM](glazewm.md)** -- Tiling window manager for Windows, i3-inspired, workspaces, resize mode

### macOS

- **[macOS](macos.md)** -- General shortcuts, Finder, Screenshots, Mission Control
- **[Rectangle](rectangle.md)** -- Window snapping with halves, thirds, quarters, and displays
- **[yabai](yabai.md)** -- Tiling window manager for macOS with BSP layout, spaces, skhd bindings

### Linux

#### Desktop Environments

- **[GNOME](gnome.md)** -- Window management, workspaces, shell navigation, accessibility
- **[KDE Plasma](kde.md)** -- Global shortcuts, window management, virtual desktops
- **[XFCE](xfce.md)** -- Global shortcuts, window management, workspaces

#### Window Managers

- **[i3](i3.md)** -- X11 tiling WM, workspaces, splits, scratchpad, resize mode
- **[Sway](sway.md)** -- Wayland i3-compatible tiling compositor, outputs, input config
- **[Hyprland](hyprland.md)** -- Dynamic Wayland tiling, animations, groups, special workspaces

#### File Managers

- **[Nautilus](nautilus.md)** -- GNOME file manager, navigation, file operations, tabs
- **[Dolphin](dolphin.md)** -- KDE file manager, split view, tabs, integrated terminal
- **[Thunar](thunar.md)** -- XFCE file manager, lightweight, custom actions, plugins

## Editors

- **[Vim](vim.md)** -- Modal editing, motions, registers, macros, windows/tabs

- **[Emacs](emacs.md)** -- Extensible editor, buffers, windows, kill ring, Dired, Org mode
- **[Helix](helix.md)** -- Selection-first modal editor, built-in LSP/Treesitter, multi-cursor

### GUI Editors

- **[VS Code](vscode.md)** -- General editing, navigation, search/replace
- **[IntelliJ IDEA](intellij.md)** -- File navigation, search, tool windows, refactoring
- **[Sublime Text](sublime.md)** -- Multi-cursor, command palette, fast editing, build systems
- **[Zed](zed.md)** -- GPU-accelerated editor, collaboration, AI assistant, LSP
