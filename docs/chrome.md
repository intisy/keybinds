### Google Chrome Keybindings

These shortcuts work in Google Chrome and most Chromium-based browsers (Edge, Brave, Vivaldi, Arc). Keybindings shown use `Ctrl` for Windows/Linux; substitute `Cmd` on macOS.

#### **Tabs**

*   **New Tab (Ctrl+T):** Open a new tab.
*   **Close Tab (Ctrl+W):** Close the current tab.
*   **Reopen Closed Tab (Ctrl+Shift+T):** Reopen the last closed tab. Can be pressed multiple times.
*   **Next Tab (Ctrl+Tab or Ctrl+Page Down):** Switch to the next tab.
*   **Previous Tab (Ctrl+Shift+Tab or Ctrl+Page Up):** Switch to the previous tab.
*   **Go to Tab (Ctrl+1-8):** Jump to tab 1 through 8.
*   **Last Tab (Ctrl+9):** Jump to the last tab.
*   **Move Tab (Ctrl+Shift+Page Up/Down):** Move the current tab left or right.
*   **Pin Tab:** Right-click the tab. No default keyboard shortcut.
*   **Mute Tab (Ctrl+M):** Mute or unmute the current tab (Chromium-based).

#### **Windows**

*   **New Window (Ctrl+N):** Open a new browser window.
*   **New Incognito Window (Ctrl+Shift+N):** Open a new incognito/private window.
*   **Close Window (Ctrl+Shift+W or Alt+F4):** Close the current window.
*   **Minimize (Super+Down):** Minimize the window (OS shortcut).
*   **Fullscreen (F11):** Toggle fullscreen mode.

#### **Navigation**

*   **Address Bar (Ctrl+L or Alt+D or F6):** Focus the address/URL bar.
*   **Go Back (Alt+Left):** Navigate to the previous page.
*   **Go Forward (Alt+Right):** Navigate to the next page.
*   **Reload (Ctrl+R or F5):** Reload the current page.
*   **Hard Reload (Ctrl+Shift+R or Ctrl+F5):** Reload, ignoring cached content.
*   **Stop Loading (Esc):** Stop loading the current page.
*   **Home Page (Alt+Home):** Go to the homepage.
*   **Scroll Down (Space or Page Down):** Scroll down one page.
*   **Scroll Up (Shift+Space or Page Up):** Scroll up one page.
*   **Go to Top (Home):** Scroll to the top of the page.
*   **Go to Bottom (End):** Scroll to the bottom of the page.

#### **Search & Find**

*   **Find on Page (Ctrl+F):** Open the find bar.
*   **Find Next (Ctrl+G or Enter):** Jump to the next match.
*   **Find Previous (Ctrl+Shift+G or Shift+Enter):** Jump to the previous match.
*   **Close Find Bar (Esc):** Close the find bar.
*   **Search Selected Text:** Select text, then drag it to the address bar or right-click > "Search".

#### **Bookmarks**

*   **Bookmark Page (Ctrl+D):** Bookmark the current page.
*   **Bookmark All Open Tabs (Ctrl+Shift+D):** Bookmark all tabs in a folder.
*   **Toggle Bookmarks Bar (Ctrl+Shift+B):** Show or hide the bookmarks bar.
*   **Bookmark Manager (Ctrl+Shift+O):** Open the bookmark manager.

#### **History & Downloads**

*   **History (Ctrl+H):** Open the history page.
*   **Downloads (Ctrl+J):** Open the downloads page.
*   **Clear Browsing Data (Ctrl+Shift+Delete):** Open the clear data dialog.

#### **Zoom**

*   **Zoom In (Ctrl+= or Ctrl+Plus):** Increase the zoom level.
*   **Zoom Out (Ctrl+-):** Decrease the zoom level.
*   **Reset Zoom (Ctrl+0):** Reset zoom to 100%.

#### **Developer Tools**

*   **Open DevTools (F12 or Ctrl+Shift+I):** Open the developer tools panel.
*   **Open Console (Ctrl+Shift+J):** Open DevTools with the console tab focused.
*   **Inspect Element (Ctrl+Shift+C):** Activate the element inspector (click an element to inspect it).
*   **View Source (Ctrl+U):** View the page source.
*   **Device Mode (Ctrl+Shift+M):** Toggle responsive/mobile device mode (inside DevTools).

#### **Other**

*   **Print (Ctrl+P):** Print the current page.
*   **Save Page (Ctrl+S):** Save the page as HTML.
*   **Open File (Ctrl+O):** Open a file from your computer.
*   **Settings (Ctrl+,):** Open Chrome settings (Chromium-based).
*   **Task Manager (Shift+Esc):** Open Chrome's built-in task manager.
*   **Toggle Caret Browsing (F7):** Enable keyboard-based cursor navigation on web pages.
