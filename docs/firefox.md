### Firefox Keybindings

Firefox keyboard shortcuts for efficient browsing. Keybindings shown use `Ctrl` for Windows/Linux; substitute `Cmd` on macOS.

#### **Tabs**

*   **New Tab (Ctrl+T):** Open a new tab.
*   **Close Tab (Ctrl+W):** Close the current tab.
*   **Undo Close Tab (Ctrl+Shift+T):** Reopen the last closed tab. Can be pressed multiple times.
*   **Next Tab (Ctrl+Tab or Ctrl+Page Down):** Switch to the next tab.
*   **Previous Tab (Ctrl+Shift+Tab or Ctrl+Page Up):** Switch to the previous tab.
*   **Go to Tab (Ctrl+1-8):** Jump to tab 1 through 8.
*   **Last Tab (Ctrl+9):** Jump to the last tab.
*   **Move Tab Left (Ctrl+Shift+Page Up):** Move the current tab to the left.
*   **Move Tab Right (Ctrl+Shift+Page Down):** Move the current tab to the right.
*   **Mute Tab (Ctrl+M):** Mute or unmute the current tab.
*   **Duplicate Tab:** No default shortcut. Use `Ctrl+L` then `Alt+Enter`.

#### **Windows**

*   **New Window (Ctrl+N):** Open a new browser window.
*   **New Private Window (Ctrl+Shift+P):** Open a new private browsing window.
*   **Close Window (Ctrl+Shift+W or Alt+F4):** Close the current window.
*   **Fullscreen (F11):** Toggle fullscreen mode.

#### **Navigation**

*   **Address Bar (Ctrl+L or Alt+D or F6):** Focus the address bar.
*   **Go Back (Alt+Left or Backspace):** Navigate to the previous page.
*   **Go Forward (Alt+Right or Shift+Backspace):** Navigate to the next page.
*   **Reload (Ctrl+R or F5):** Reload the current page.
*   **Hard Reload (Ctrl+Shift+R or Ctrl+F5):** Reload, bypassing the cache.
*   **Stop Loading (Esc):** Stop loading the page.
*   **Home Page (Alt+Home):** Go to the homepage.
*   **Scroll Down (Space or Page Down):** Scroll down one page.
*   **Scroll Up (Shift+Space or Page Up):** Scroll up one page.
*   **Go to Top (Home):** Scroll to the top.
*   **Go to Bottom (End):** Scroll to the bottom.

#### **Search & Find**

*   **Find on Page (Ctrl+F):** Open the find bar.
*   **Find Again (Ctrl+G or F3):** Jump to the next match.
*   **Find Previous (Ctrl+Shift+G or Shift+F3):** Jump to the previous match.
*   **Quick Find (/):** Start quick find (type to search, auto-closes).
*   **Quick Find Links Only ('):** Quick find but only matches link text.
*   **Close Find Bar (Esc):** Close the find bar.
*   **Search Bar (Ctrl+K or Ctrl+E):** Focus the search bar / address bar for search.

#### **Bookmarks**

*   **Bookmark Page (Ctrl+D):** Bookmark the current page.
*   **Bookmark All Tabs (Ctrl+Shift+D):** Bookmark all open tabs.
*   **Toggle Bookmarks Toolbar (Ctrl+Shift+B):** Show or hide the bookmarks toolbar.
*   **Library/Bookmarks Manager (Ctrl+Shift+O):** Open the bookmarks library.
*   **Bookmarks Sidebar (Ctrl+B):** Toggle the bookmarks sidebar.

#### **History & Downloads**

*   **History Sidebar (Ctrl+H):** Toggle the history sidebar.
*   **Downloads (Ctrl+J):** Open the downloads panel.
*   **Clear Recent History (Ctrl+Shift+Delete):** Open the clear history dialog.
*   **Recently Closed Windows/Tabs:** Available in `History > Recently Closed`.

#### **Zoom**

*   **Zoom In (Ctrl+= or Ctrl+Plus):** Increase zoom.
*   **Zoom Out (Ctrl+-):** Decrease zoom.
*   **Reset Zoom (Ctrl+0):** Reset to 100%.

#### **Developer Tools**

*   **Open DevTools (F12 or Ctrl+Shift+I):** Toggle the developer tools.
*   **Web Console (Ctrl+Shift+K):** Open the web console directly.
*   **Inspector (Ctrl+Shift+C):** Activate the element picker.
*   **Debugger (Ctrl+Shift+Z):** Open the JavaScript debugger.
*   **Network Monitor (Ctrl+Shift+E):** Open the network tab.
*   **Style Editor (Shift+F7):** Open the style editor.
*   **View Source (Ctrl+U):** View the page source.
*   **Responsive Design Mode (Ctrl+Shift+M):** Toggle responsive design mode.

#### **Firefox-Specific**

*   **Reader Mode (F9):** Toggle Reader View for articles.
*   **Screenshot (Ctrl+Shift+S):** Take a screenshot (built-in Firefox tool).
*   **Page Info (Ctrl+I):** View page information (security, media, permissions).
*   **Add-ons Manager (Ctrl+Shift+A):** Open the extensions/add-ons page.
*   **Toggle Caret Browsing (F7):** Navigate pages with a text cursor.
*   **About Config:** Type `about:config` in the address bar for advanced settings.
