### Dolphin File Manager Keybindings

Dolphin is the default file manager for KDE Plasma. It features split views, tabs, integrated terminal, and extensive keyboard navigation.

#### **Navigation**

*   **Go Back (Alt+Left):** Navigate to the previous directory.
*   **Go Forward (Alt+Right):** Navigate to the next directory.
*   **Go Up (Alt+Up):** Navigate to the parent directory.
*   **Go Home (Alt+Home):** Navigate to the home directory.
*   **Address Bar (Ctrl+L):** Focus the path/address bar for manual input.
*   **Breadcrumb Bar (Esc):** Switch back to the breadcrumb navigation bar.
*   **Filter Bar (Ctrl+I):** Open the filter bar to filter files by name.
*   **Open Terminal (F4):** Toggle the built-in terminal panel at the bottom.

#### **File Operations**

*   **Copy (Ctrl+C):** Copy selected files.
*   **Cut (Ctrl+X):** Cut selected files.
*   **Paste (Ctrl+V):** Paste files.
*   **Undo (Ctrl+Z):** Undo the last file operation.
*   **Delete (Delete):** Move selected files to the trash.
*   **Delete Permanently (Shift+Delete):** Permanently delete selected files.
*   **Rename (F2):** Rename the selected file.
*   **New Folder (F10):** Create a new folder.
*   **Open (Enter):** Open the selected file or directory.
*   **Open With (Alt+Enter then choose):** Select an application to open the file.
*   **Properties (Alt+Enter):** Show file properties.
*   **Select All (Ctrl+A):** Select all files.
*   **Invert Selection (Ctrl+Shift+A):** Invert the current selection.

#### **Tabs & Split View**

*   **New Tab (Ctrl+T):** Open a new tab.
*   **Close Tab (Ctrl+W):** Close the current tab.
*   **Next Tab (Ctrl+Tab or Ctrl+Page Down):** Switch to the next tab.
*   **Previous Tab (Ctrl+Shift+Tab or Ctrl+Page Up):** Switch to the previous tab.
*   **Duplicate Tab (Ctrl+D):** Duplicate the current tab.
*   **Split View (F3):** Toggle split view (two directory panels side by side).
*   **Focus Other Panel (Tab):** Switch focus between split view panels.

#### **View**

*   **Icons View (Ctrl+1):** Switch to icons/grid view.
*   **Compact View (Ctrl+2):** Switch to compact view.
*   **Details View (Ctrl+3):** Switch to detailed list view.
*   **Show Hidden Files (Ctrl+H or Alt+.):** Toggle hidden files visibility.
*   **Preview Panel (F11):** Toggle the preview/information panel.
*   **Zoom In (Ctrl+= or Ctrl+Plus):** Increase icon size.
*   **Zoom Out (Ctrl+-):** Decrease icon size.
*   **Refresh (F5):** Refresh the directory listing.

#### **Search**

*   **Search (Ctrl+F):** Open the search bar.
*   **Close Search (Esc):** Close the search bar and return to browsing.

#### **Customization**

*   Dolphin's shortcuts can be customized via **Settings > Configure Keyboard Shortcuts**.
*   Many features can also be accessed through the right-click context menu.
*   Services (custom context menu actions) can be installed via **Settings > Configure Dolphin > Context Menu > Download New Services**.
